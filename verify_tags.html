Verificación de Tags en Consola

Abre la consola de Chrome (F12) en www.entersys.mx y ejecuta estos comandos:

1. Verificar GA4

// Debería mostrar el dataLayer con eventos de GA4
console.log(window.dataLayer);

// Buscar configuración de GA4
window.dataLayer.filter(item => item[0] === 'config' || item.event === 'gtm.js');

2. Verificar Facebook Pixel

// Debería devolver true si está cargado
console.log(typeof fbq !== 'undefined');

// Ver la cola de eventos
console.log(window.fbq?.queue || 'No cargado');

3. Verificar LinkedIn Insight Tag

// Debería devolver tu Partner ID
console.log(window._linkedin_partner_id);

// Verificar que el script se cargó
console.log(typeof window.lintrk !== 'undefined');
